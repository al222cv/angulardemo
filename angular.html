Varför finns AngularJS?

För att lösa problemet med att HTML är gjort för statiska sidor och inte dynamiska applikationer

Varför AngularJS istället för något annat ramverk?

För att andra ramverk hanterar HTML:s tillkortakommanden genom att abstrahera bort HTML, CSS och/eller JavaScript.

Angular "extendar" istället HTML.

Varför har AngularJS blivit så stort och populärt så snabbt?

  • För att AngularJS är deklarativt
  • För att AngularJS guidar/säger åt dig hur du ska strukturera och koda din kod

Varför har AngularJS blivit så 'hatat' av så många?

  • För att AngularJS är deklarativt
  • För att AngularJS guidar/säger åt dig hur du ska strukturera och koda din kod
